# uart_prog
Programmer spi flash 25xx, i2c eeprom 24xx and microwire eeprom 93xx on arduino.

 Проект позволяет сделать из arduino nano/uno (а при желании - из любого микроконтроллера, имеющего на борту UART,SPI,I2C и хотя бы 1Кб ОЗУ) программатор SPI-Flash 25-й серии, I2C-EEPROM 24-й серии и Microwire EEPROM 93-й серии.
